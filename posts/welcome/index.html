It's also a good way to spot unapproved verbs being used. In that case, the verb is bold, and the noun is not. :D Previous versions of this post post had Do-Something instead of Get-Something. So ladies and gentleman, always look at the approved list in Get-Verb when naming your functions.

The name … ?

For a long time I was having this idea “How Posh”, not like “how to do powershell”, but more like posh/fancy/snobbish … And I've seen people use posh as a short name for PowerShell. And some brainfart later…

It stuck in my head all this time. :D

But now that the PowerShell core executable is named pwsh instead, because the posh executable was already taken on *nix systems, the title doesn't sound universal anymore. So I mixed in some regex to refer to both.

I did decide to use pwsh for the url, so it would show my commitment to PowerShell core.
